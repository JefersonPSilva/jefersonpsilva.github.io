  • Ajax
    O que é?

    AJAX significa Asynchronous JavaScript e XML. É o uso do objeto XMLHttpRequest para se comunicar com os scripts do lado do servidor. Ele pode enviar bem como receber informações em uma variedade de formatos, incluindo JSON, XML, HTML, e até mesmo arquivos de texto. Porém a característica mais atraente do AJAX, é a sua natureza "assíncrona", o que significa que ele pode fazer tudo isso sem a necessidade de atualizar a página.

    Experiência:

    Manipulação de objetos JSON, manipulação de informações do Banco de Dados proveniente do PHP em Server-Side para o Client-Side, manipulação de dados Geoespaciais para aplicação em mapas interativos.

HTML

  • HTML, XHTML
    O que é?

  • HTML5
    O que é?

É um conjunto maior de tecnologias que permite o desenvolvimento de aplicações e web sites mais diversos e poderosos, com melhor semântica para os interpretadores dos Browser, é que possibilita melhor posicionamento com tecnícas de SEO(Search Engine Optimization).
